# Inbox

- Runtime `fetch-archive` DNS-rebinding TOCTOU: `validate_fetch_url` resolves the host and rejects internal addresses, then `reqwest` re-resolves independently at connect time, so a host that resolves to a public IP during validation and an internal IP at connect could slip past the SSRF screen. Inherent to BE-INPUT-007's prescribed resolve-then-block design (its Verification explicitly accepts resolution-time blocks), so not a review violation — but a hardened form (custom resolver / pinned socket addr passed to the connector, or connect-time re-check) would close the window. Low priority. Surfaced 2026-07-11 during the 0.18.0 fix-verification pass.

- Architectural exploration: re-frame the runtime's LLM extension points (`writeCode`, `writeSpecBody`, `assessSpecQuality`, future multi-turn points) as named Anthropic-style Skills the host loads at the seam, rather than ad-hoc JSON envelopes. Potential benefits: structural cache anchoring (Skills are a natural cache boundary); third-party hosts integrate against an emerging Skills protocol instead of govern-specific JSON; `constitution-excerpts` becomes a bundled resource rather than an inline string array. Speculative — depends on Anthropic's Skills protocol stabilizing and is a larger redesign than 022's current scope. Revisit after the writeCode payload-bundling scenario on 022 ships and the cache-anchored shape proves out the pattern. Surfaced 2026-05-19 during runtime-improvement investigation. **On hold per user 2026-07-11.**
- Runtime dependency currency: bump rmcp from the 1.x pin (Cargo.toml `rmcp = "1"`, resolves 1.8.0) to 2.x (2.2.0 available). Breaking major — the MCP server's 41 `#[tool]` methods, ToolRouter, ServiceExt, and transport::stdio all sit on rmcp's macro/router API that reworked between 1.x and 2.x, so it needs a port + full test pass (incl. tests/mcp.rs) + its own gvrn-v tag (the next minor after the current release). Non-urgent: 1.8.0 is functional, no advisory; cargo just notes the newer major on install. Read rmcp's 2.x migration guide to scope before starting. Surfaced 2026-07-11 post-0.18.0 release.
