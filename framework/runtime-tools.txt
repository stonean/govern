# Source of truth: framework/constitution.md §runtime-boundary
#
# Each non-blank, non-comment line is one tool name (matching MCP tool names
# or runtime binary names). The list is consumed by:
#
#   - .github/workflows/markdown-only-pipeline.yml (asserts no entry is on PATH)
#   - scripts/lint-tool-coverage.sh              (verifies references in
#                                                  framework/commands/*.md
#                                                  are wrapped in a graceful
#                                                  fallback)
#
# Tool names are case-sensitive exact-match strings. Names are bare
# `<verb>-<noun>` strings — server-level namespacing (`gvrn`) is
# supplied by each host's MCP server registration, producing the
# host-specific wire identifier (Claude Code: `mcp__gvrn__<verb>-<noun>`;
# Auggie / Antigravity: `mcp:gvrn:<verb>-<noun>`). The bare names are
# also the `gvrn <name>` subcommand identifiers.
